# 影片心率和 Experience Cloud ID 服務 {#video-heartbeat-and-the-experience-cloud-id-service}

視訊心率需要Experience Cloud ID服務。在測量 JavaScript 或 Flash 上的視訊時，在您的網站頁面上與視訊播放中實作 ID 服務，可確保在訪客檢視視訊時不會個別計算訪客人次。
