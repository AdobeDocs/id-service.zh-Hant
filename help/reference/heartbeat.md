# 影片心率和 Experience Cloud Identity Service {#video-heartbeat-and-the-experience-cloud-id-service}

影片心率必須與 Experience Cloud Identity Service 搭配使用。在測量 JavaScript 或 Flash 上的視訊時，在您的網站頁面上與視訊播放中實作 ID 服務，可確保在訪客檢視視訊時不會個別計算訪客人次。
